Module Documentation
Overview:
This module provides tools for working with CSV data. It includes a DataProcessor class for loading, filtering, and exporting data, as well as a merge_records function for combining two datasets.
DataProcessor Class:

The DataProcessor class takes a file path and delimiter as initialization arguments.
It provides methods:
filter_rows: Filters rows based on specified criteria.
export: Exports the data to a new CSV file.



merge_records Function:

The merge_records function merges two lists of dictionaries based on a common key.

Usage Examples:
```python
Create a DataProcessor object
processor = DataProcessor(file_path="input.csv", delimiter=";")
Filter rows based on age range and activity status
filtered = processor.filter_rows(min_age=18, max_age=30, include_inactive=False)
Export the filtered data to a new CSV file
processor.export("filtered_output.csv")
Merge two datasets based on the 'id' field
merged = merge_records(records_a=[{"id": 1, "name": "Alice"}],
records_b=[{"id": 1, "score": 95}],
key="id")
```
Expected Output:

The filter_rows call returns a list of rows where the age is between 18 and 30, and the activity status is not inactive.
The export method writes the filtered data to the specified file path.
The merge_records function returns a new list of dictionaries with combined information from both datasets.
