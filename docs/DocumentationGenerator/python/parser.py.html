OVERVIEW:
This module provides functionality for parsing and manipulating data from Python files. It includes functions for parsing functions, classes, and modules from AST trees.
USAGE EXAMPLES:
```python
Parse a single file
tree = ast.parse(open('file.py').read())
functions = parseFunctionsFromTree(tree)
Parse a directory
module_data = parseDirectory('directory')
Parse a module
module_docstring = parseDocstringFromModule(module)
```
EXPECTED OUTPUT:

The parseFunctionsFromTree function returns a list of Function objects representing the functions defined in the AST tree.
The parseDirectory function returns a dictionary mapping file paths to AST trees.
The parseDocstringFromModule function returns the docstring of the AST module.
