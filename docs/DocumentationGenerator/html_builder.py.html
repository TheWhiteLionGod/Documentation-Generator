OVERVIEW
This module provides a HtmlBuilder class for creating HTML tags and a collection of helper functions. The HtmlBuilder class allows users to easily build HTML content by chaining methods like createH4, createParagraph, and createLink.
USAGE EXAMPLES
```python
Create an HtmlBuilder object
builder = HtmlBuilder(generateInitalHtml=True)
Build an H4 tag with some content
builder.createH4(contents="My Heading")
Build a paragraph tag with some content
builder.createParagraph(contents="This is a paragraph.")
Build a link tag with a target URL
builder.createLink(contents="Click here", link="example.com")
Build the HTML content
html = builder.build()
Print the HTML content
print(html)
```
EXPECTED OUTPUT
The code above will output a valid HTML document containing an H4 tag, a paragraph tag, and a link tag. The specific HTML content will depend on the input parameters provided to the createH4, createParagraph, and createLink methods.
